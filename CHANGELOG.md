## [0.11.0] - 2026-06-01

### Added

- **create:** Add `--install-python` flag for lazy Python install ([#115](https://github.com/ai-screams/scoop-uv/pull/115))
- **cli:** Add `scoop run <env> -- <cmd>` for one-shot execution without activation ([#115](https://github.com/ai-screams/scoop-uv/pull/115))
- **cli:** Add `scoop status` to summarise the current env (Active/Configured/System/None) ([#115](https://github.com/ai-screams/scoop-uv/pull/115))

- **manifest:** Add `.scoop.toml` project manifest with parent-walk discovery ([#116](https://github.com/ai-screams/scoop-uv/pull/116))
- **cli:** Add `scoop sync` to reconcile the active env with `.scoop.toml` ([#116](https://github.com/ai-screams/scoop-uv/pull/116))
- **cli:** Add `scoop export <env>` to snapshot envs as JSON (schema v1) ([#117](https://github.com/ai-screams/scoop-uv/pull/117))
- **cli:** Add `scoop import <file>` to restore exported envs ([#117](https://github.com/ai-screams/scoop-uv/pull/117))
- **cli:** Add `scoop clone <src> <dst>` to duplicate an env in-place ([#117](https://github.com/ai-screams/scoop-uv/pull/117))
- **devcontainer:** Add VS Code / Codespaces dev container with rust:1-bookworm + nextest/llvm-cov/mutants/uv ([#118](https://github.com/ai-screams/scoop-uv/pull/118))
- **bench:** Add Criterion benchmarks (parsing/validation/path_lookup) with `make bench` ([#118](https://github.com/ai-screams/scoop-uv/pull/118))

### Changed

- **migrate:** Parallelise `migrate all` with rayon for multi-core scaling; dry-run + single-env stay sequential ([#117](https://github.com/ai-screams/scoop-uv/pull/117))
- **ci:** Replace pyenv-only integration test with matrix over pyenv/conda/venvwrapper, per-shard BuildKit cache scoping ([#118](https://github.com/ai-screams/scoop-uv/pull/118))
- **ci:** Add Criterion regression gate publishing to `gh-pages` (alert-threshold 150%) ([#118](https://github.com/ai-screams/scoop-uv/pull/118))
- **ci:** Add `cargo-mutants --in-diff` PR gate (full mutation run weekly) ([#118](https://github.com/ai-screams/scoop-uv/pull/118))

- **deps:** Bump github-actions group across 1 directory with 3 updates ([#121](https://github.com/ai-screams/scoop-uv/pull/121))

## [0.7.0] - 2026-02-06

### Added

- Implement N-1 MSRV policy with full automation ([#69](https://github.com/ai-screams/scoop-uv/pull/69))

### Changed

- **MSRV**: Adopted N-1 MSRV policy (current: 1.85, required by Edition 2024)
  - Support current stable Rust + one previous version (~6 week lag)
  - MSRV bumps are non-breaking for binary users per [Cargo RFC 3537](https://rust-lang.github.io/rfcs/3537-msrv-resolver.html)
  - CI now tests both MSRV (1.85) and stable Rust versions
  - Added cargo-msrv verification for automated MSRV checking
  - See [README.md](README.md#minimum-supported-rust-version-msrv-) for full policy details
